# Stuttgart-Gophers-Gotron

A counter example used to showcase Gotron in  live coding session
during presentation on Stuttgart-Gophers meetup group.

## Prerequisites
go, nodejs, npm

## Run 
When *make* is installed
```
make install 
make build
make run
```

Without *make*
```
cd ui && npm i
cd ui && webpack
go run main.go
```
